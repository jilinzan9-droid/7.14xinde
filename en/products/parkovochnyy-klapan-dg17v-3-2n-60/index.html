Hydraulic Directional Control Valves

DG17V-3-2N-60Lever-Operated Directional Control Valve

The Vickers by Danfoss DG17V-3-2N-60 is a single-stage manual directional valve for industrial hydraulic circuits. Its Type 2 spool uses a no-spring detent and mounts on an ISO 4401-03 / NFPA D03 interface.

Brand: Vickers by DanfossItem: 02-127344Interface: ISO 4401-03

Product Overview

Manual Flow Direction Control

Designed to manually direct hydraulic flow where electrical actuation is not required. Final suitability must be confirmed against the circuit diagram, fluid, pressure and tank-line conditions.

350 barRated system pressure
75 L/minMaximum flow, 20 US gpm
D03 / NG6ISO 4401-03 mounting interface
2NType 2 spool, no spring, detented

Hydraulic function

The manual lever selects the valve spool position to control the direction, start or stop of petroleum-oil flow in circuits operating a hydraulic cylinder or motor.

Configuration control

The complete model code defines the actuation, pressure class, mounting interface, spool and spring arrangement. Confirm the full code and manufacturer item number before replacement.

Photo and model do not match

This listing is for DG17V-3-2N-60, item 02-127344. The Russian source photograph visibly shows DG17V-3-2A-60, item 02-127342. “N” is no spring with detent; “A” is spring offset to cylinder A. These configurations must not be treated as interchangeable.

Compatibility confirmation

Send the complete nameplate, hydraulic symbol, mounting face, system pressure and flow, tank-line pressure and required lever orientation before quotation.
